# Сессия: Stop-хук для метрик токенов

**Дата:** 2026-05-28  
**Тип:** feat (глобальная конфигурация)

## Цель

Автоматически дописывать метрики токенов/стоимости в конец session-файлов `docs/sessions/*.md` после их создания.

## Результат

- Создан скрипт `~/.claude/scripts/append-session-metrics.ps1`
  - Читает `transcript_path` из stdin (Stop-хук payload)
  - Парсит JSONL-транскрипт: суммирует input/cache_read/cache_write/output токены
  - Находит `docs/sessions/*.md`, изменённый < 120 сек назад
  - Дописывает секцию `## Метрики сессии` с моделью, токенами, контекстом%, стоимостью
  - Защита от двойной записи и тихий exit 0 при любых ошибках
- Зарегистрирован Stop-хук в `~/.claude/settings.json` (глобальный)
- Pipe-test прошёл; обнаружен и починен баг: файл нужно сохранять с UTF-8 BOM для PS 5.1
- Закоммичены спек и план в репозиторий
