# Сессия: Выделение выбранного дрона и шкалы ресурсов на поле

**Дата/время:** 2026-05-29 ~11:50  
**Тип:** feature  
**Статус:** ✅ завершено

## Цель

Добавить визуальную обратную связь на игровом поле:
1. Выделить выбранного дрона кольцом-обводкой.
2. Показать компактные шкалы энергии и загрузки над каждым дроном.

Раньше состояние дрона можно было узнать только через правую панель (DroneInspector), что затрудняло отслеживание стаи.

## Результаты

### Реализовано

**DroneSprite.ts** — расширен Container тремя новыми Phaser-объектами:
- `_selectionRing` — дугообразное кольцо (Arc, радиус 16 px, белый stroke толщиной 2 px, без заливки)
- `_energyBar` — Graphics-полоска энергии (28×3 px, голубая #00d4ff)
- `_loadBar` — Graphics-полоска загрузки (28×3 px, зелёная #00ff88)

Новый API:
- `setSelected(boolean)` — переключает видимость кольца
- `updateStats(energyRatio, loadRatio)` — перерисовывает шкалы; кеширует ratio для оптимизации

**GameScene.ts** — интеграция в syncSprites():
- Подключено чтение `selectedDroneId` из useGameStore
- Каждому дрону передаются значения энергии и загрузки из World (`getComponent`)
- На каждом тике вызываются `setSelected()` и `updateStats()`

## Дизайн-решения

- **Кольцо статичное** — пользователь выбрал простую визуализацию без пульсации
- **Шкалы всегда видны** — компактный режим без подписей чисел
- **Z-порядок Container** — `_light, _selectionRing, _body, _energyBar, _loadBar` (шкалы над всем)
- **Оптимизация Graphics** — шкалы перерисовываются только при изменении ratio
- **Координаты локальные** — шкалы следуют за дроном автоматически при интерполяции позиции
